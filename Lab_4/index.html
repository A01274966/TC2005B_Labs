1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

JavaScript es un lenguaje recomendado para el desarrollo de aplicaciones y sitios web. Es un lenguaje por asi decirlo "creativo" y por lo tanto, esta diseñado para el desarrollo front-end y con la llegada de NodeJS ha hecho posible que los desarrolladores utilicen JavaScript en back-end tambien.

Java es un lenguaje de back-end recomendado para aplicaciones independeientes. Se considera que es el idioma nativo de las aplicaciones Android. Se recomienda para desarrollar aplicaiones robustas. Se dice que Java es un lenguaje detallado, por lo que a primera vista puede parecer un poco más fácil de utilizar.
